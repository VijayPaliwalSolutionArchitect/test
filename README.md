# MegaMart E-commerce App

## Overview
MegaMart is a full-featured e-commerce platform including user management, product management, cart, orders, reviews, blogs, and admin dashboard.

## Tech Stack
- **Backend:** Node.js, Express, MongoDB
- **Frontend:** React 19, TailwindCSS, shadcn/ui
- **API Docs:** Swagger/OpenAPI
- **Auth:** JWT, BCrypt
- **Dev Tools:** ESLint, Prettier

## Main Features
- Product catalog, categories, brands
- Cart, checkout, order management
- Review & rating system
- Blog & content module
- Admin dashboard

## Menu Structure
- Home
- Products & Categories
- Blogs
- Cart
- Orders
- Admin (products, orders, users, blogs)

## Quick Start

### Backend
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
